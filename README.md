# Cricket Analytics Dashboard - T20 World Cup 2022

## Project Overview

This project analyzes T20 World Cup 2022 cricket data and helps identify the best playing XI based on player performance metrics.

## Tools & Technologies

* Python
* Pandas
* Power BI
* DAX
* JSON Data Processing

## Project Workflow

1. Collected cricket match data in JSON format.
2. Cleaned and transformed data using Python and Pandas.
3. Created fact and dimension tables.
4. Built data models and relationships in Power BI.
5. Created DAX measures for batting and bowling analysis.
6. Applied player selection criteria.
7. Developed interactive dashboards for player comparison and team selection.

## Dashboard Features

* Power Hitters Analysis
* Anchors Analysis
* Finishers Analysis
* All-Rounders Analysis
* Specialist Fast Bowlers Analysis
* Best Playing XI Selection

## Dataset

T20 World Cup 2022 match, batting, bowling, and player statistics.
